# Security Policy

This is a personal repository used for playing around with Laravel. 

## Reporting a Vulnerability

This is a personal project and not intended for production use. If you discover a security vulnerability:

1. **Do not disclose the vulnerability publicly**
2. **Do not expect a response** as this is a personal project

## Note

This repository is maintained by a single developer for personal use and experimentation. There are no formal security processes or response timelines in place.
